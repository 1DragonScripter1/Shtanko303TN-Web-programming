Мої гіперпосилання

GitHub
GitHub - найбільший вебсервіс для спільної розробки IT-проєктів та їхнього хостингу.
Stack Overflow
Stack Overflow - система запитань і відповідей на тему програмування. Найкраще місце, щоб знайти відповідь на складне питання.
Codewars
Codewars - сайт для практики програмування з різними рівнями складності.
LeetCode
LeetCode - платформа для підготовки до технічних співбесід з великою базою задач.
freeCodeCamp
freeCodeCamp - некомерційна організація, яка пропонує безкоштовні онлайн-курси з веб-розробки.
Coursera
Coursera - онлайн-платформа, яка пропонує курси від провідних університетів та компаній світу.
Djinni
Djinni - сервіс анонімного пошуку роботи для IT-спеціалістів в Україні.
DOU
DOU - найбіліша спільнота розробників України.
